# AI-Based Waste Segregation Assistant

This project is an AI-based waste segregation system developed using Python and Flask.  
It helps users classify household waste into Wet, Dry, or Hazardous categories and provides proper disposal guidance.

## Features
- Web-based input and output interface
- REST API for waste classification
- Dataset-driven rule-based AI logic
- Supports sustainable waste management (SDG 12)

## Technologies Used
- Python
- Flask REST API
- Rule-Based AI Logic
- CSV Dataset

## How to Run
```bash
pip install -r requirements.txt
python app.py
